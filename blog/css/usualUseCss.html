  z-index: 1;
  text-align: center;
  -ms-transform: scale(1.185);
  transform: scale(1.185);
  background: rgba(0, 0, 0, 0.6);
  transition: all 0.3s ease-in-out 0s;
  opacity: 0; //核心代码
  pointer-events: none;//核心代码
  &.show {
    transition-duration: 0.3s;
    -ms-transform: scale(1);
    transform: scale(1);
    opacity: 1;//核心代码
    pointer-events: auto;//核心代码
  }
}
...
</style>
也是一个很简单的弹窗布局,因此多余代码省略了,这里重点只讨论下效果的核心代码实现
opacity: 透明度,这个属性一定是必要的,默认全透明,当打开弹窗默认完全不透明
pointer-events: 是css3中的一个属性,支持的值也是相当丰富,不过大多数都是与svg有关的,一般也用不到,在这里我们只需要重点关注下 pointer-events:none 这个属性值就ok. pointer-events:none从字面意思就可以get到其中心思想,大概就是让点击事件(鼠标事件)拜拜.这里的弹窗效果之所以能够实现,该属性一定是c位出道.
  • 想要了解更多关于pointer-events信息,推荐这篇文章 CSS3 pointer-events:none应用举例及扩展

# loadProgress(仿h5顶部加载进度条)

<!--loadProgress.vue-->
<template>
  <div class="load-progress" :class="loadProgress!=0?'show':'hide'" :style="[{top:CustomBar+'px'}]">
    <div
      class="load-progress-bar bg-green"
      :style="[{transform: 'translate3d(-' + (100-loadProgress) + '%, 0px, 0px)'}]"
    ></div>
  </div>
</template>

<script>
export default {
  props: {},
  components: {},
  data() {
    return {
      loadProgress: 0
    };
  },
  methods: {
    LoadProgress(e) {
      this.loadProgress = this.loadProgress + 3;
      if (this.loadProgress < 100) {
        setTimeout(() => {
          this.LoadProgress();
        }, 100);
      } else {
        this.loadProgress = 0;
      }
    }
  },
  computed: {},
  watch: {},
  created() {},
  mounted() {
    this.LoadProgress();
  }
};
</script>

<style scoped lang="scss">
